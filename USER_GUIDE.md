## 1. 这个工具能帮我做什么？

如果你在闲鱼上卖东西，每天需要重复做很多事情：

- 拍照、写标题、写描述、上架商品
- 每天点"擦亮"让商品排名靠前
- 调整价格、下架卖完的东西、重新上架
- 查看今天有多少人看了、多少人想要

**这个工具可以帮你自动完成以上所有操作。**

你只需要在电脑的浏览器里打开一个网页，点几下按钮，工具就会自动帮你操作闲鱼。
它还可以用 AI（人工智能）帮你自动写出吸引人的商品标题和描述。

## 2. 我需要准备什么？

| 你需要 | 具体说明 |
|--------|---------|
| 一台电脑 | Windows 10/11 或 macOS 都可以 |
| 能上网 | 需要连接互联网 |
| 闲鱼账号 | 你平时用的闲鱼账号，能正常登录即可 |
| 大约 30 分钟 | 第一次安装大约需要 30 分钟，以后每次使用只需要几秒钟启动 |

> 不需要任何编程知识，不需要懂代码，按照下面的步骤一步步来就行。

## 3. 第一步：把工具装到电脑上

### 方式一：一键安装（推荐新手使用）

在安装之前，你需要先安装两个免费的基础软件。就像手机上装 App 一样简单：

#### 3.1 安装 Python（工具运行需要它）

1. 打开浏览器，访问 https://www.python.org/downloads/
2. 点击页面上黄色的大按钮 **"Download Python 3.x.x"**
3. 下载完成后，双击打开安装文件
4. **重要**：在安装界面的最底部，一定要勾选 **"Add Python to PATH"** 这个选项
5. 然后点击 **"Install Now"** 开始安装
6. 等待安装完成，点击 **"Close"** 关闭

#### 3.2 安装 Node.js（网页界面需要它）

1. 打开浏览器，访问 https://nodejs.org/
2. 点击左侧写着 **"LTS"** 的绿色大按钮下载
3. 下载完成后，双击打开安装文件
4. 一路点 **"Next"（下一步）** 直到安装完成

#### 3.3 下载本工具

1. 打开浏览器，访问 https://github.com/G3niusYukki/xianyu-openclaw
2. 点击绿色的 **"Code"** 按钮
3. 点击 **"Download ZIP"**
4. 下载完成后，找到这个 ZIP 文件（通常在"下载"文件夹里）
5. **右键点击** 这个 ZIP 文件，选择 **"解压到当前文件夹"**（Windows）或直接双击（macOS）
6. 你会得到一个名为 `xianyu-openclaw-main` 的文件夹

#### 3.4 运行安装脚本

**Windows 用户：**
1. 双击打开 `xianyu-openclaw-main` 文件夹
2. 找到 `install.bat` 这个文件，双击运行它
3. 会弹出一个黑色的窗口，显示安装进度
4. 看到 **"安装完成！"** 的提示后，按任意键关闭窗口

**macOS 用户：**
1. 打开"终端"应用（在"启动台" -> "其他"里可以找到）
2. 输入 `cd `（cd 后面有个空格），然后把 `xianyu-openclaw-main` 文件夹拖到终端窗口里，按回车
3. 输入 `bash install.sh`，按回车
4. 等待安装完成

### 方式二：Docker 安装（适合有服务器的用户）

如果你有自己的服务器，或者你身边有懂技术的人帮忙，可以用 Docker 一键部署：

1. 确保服务器上已安装 Docker
2. 将项目文件上传到服务器
3. 在项目目录下执行 `docker compose up -d`
4. 访问 `http://服务器IP:8501` 即可使用

## 4. 第二步：获取闲鱼登录凭证（Cookie）

> **什么是 Cookie？** 
> 简单理解：Cookie 就像你的"临时通行证"。当你登录闲鱼后，浏览器会保存一串文字，
> 用来证明"我已经登录了"。我们需要把这串文字告诉工具，工具才能代替你操作闲鱼。

### 获取步骤（以 Chrome 浏览器为例）

1. 打开 Chrome 浏览器
2. 在地址栏输入 `https://www.goofish.com` 并按回车，打开闲鱼网页版
3. 登录你的闲鱼账号（手机扫码或密码登录都可以）
4. 登录成功后，同时按下键盘上的 **F12** 键（如果是 macOS，按 **Fn + F12**）
   - 这时候浏览器右侧或底部会弹出一个面板，这就是"开发者工具"
5. 在弹出的面板顶部，找到并点击 **"Network"**（网络）这个标签
6. 按键盘上的 **F5** 刷新一下页面
7. 在 Network 面板左侧会出现很多行，随便点击其中一行
8. 在右侧找到 **"Request Headers"**（请求头）部分
9. 往下翻，找到 **"Cookie:"** 这一项
10. Cookie 后面有一长串文字，**全部选中并复制**（Ctrl+C 或 Command+C）

> 提示：Cookie 通常很长，可能有好几行。请确保全部复制，不要只复制一部分。

### 注意事项

- Cookie 有有效期，通常 **7-30 天后会过期**
- 过期后需要重新登录闲鱼网页版，重复上面的步骤获取新的 Cookie
- **不要把 Cookie 分享给任何人**，它相当于你的登录密码

## 5. 第三步：填写配置信息

安装完成后，你需要告诉工具你的闲鱼 Cookie 和 AI 密钥。

### 5.1 配置闲鱼 Cookie

1. 在 `xianyu-openclaw-main` 文件夹中，找到 `.env` 文件
   - 如果看不到这个文件，说明系统隐藏了它
   - **Windows**：在文件夹顶部菜单点击"查看"，勾选"隐藏的项目"
   - **macOS**：在文件夹中按 `Command + Shift + .`（句号键）显示隐藏文件
2. 用记事本（Windows）或文本编辑器（macOS）打开 `.env` 文件
3. 找到这一行：

```
XIANYU_COOKIE_1=your_cookie_here
```

4. 把 `your_cookie_here` 替换成你刚刚复制的 Cookie
5. 如果你有第二个闲鱼账号，同样替换 `XIANYU_COOKIE_2=` 后面的内容
6. 保存文件并关闭

### 5.2 配置 AI 功能（可选，但推荐）

AI 功能可以帮你自动生成吸引人的商品标题和描述。如果你不需要，可以跳过这一步。

目前支持两种 AI 服务，任选其一：

**选择 DeepSeek（推荐，价格便宜）：**
1. 打开浏览器，访问 https://platform.deepseek.com/
2. 注册一个账号
3. 在控制台找到"API Keys"，创建一个新的密钥
4. 复制这个密钥
5. 在 `.env` 文件中，把 `DEEPSEEK_API_KEY=sk-...` 中的 `sk-...` 替换成你的密钥

**选择 OpenAI：**
1. 打开浏览器，访问 https://platform.openai.com/
2. 注册账号并创建 API Key
3. 在 `.env` 文件中，把 `OPENAI_API_KEY=sk-...` 中的 `sk-...` 替换成你的密钥

> 两个 AI 服务只需要配置一个就行。不配置也能使用工具，只是无法自动生成标题和描述，需要手动填写。

## 6. 第四步：启动工具

### 启动方法

**Windows 用户：**
1. 在 `xianyu-openclaw-main` 文件夹中找到 `start.bat`
2. 双击运行它
3. 会弹出一个黑色窗口，显示启动信息
4. 等待几秒钟后，浏览器会自动打开工具的网页界面

**macOS 用户：**
1. 打开"终端"
2. 进入项目目录（和安装时一样）
3. 输入 `bash start.sh`，按回车

**Docker 用户：**
```
docker compose up -d
```

### 访问工具

启动后，打开你的浏览器，在地址栏输入：

```
http://localhost:8501
```

按回车后，你就能看到工具的主界面了。

> 如果浏览器没有自动打开，手动输入上面的地址即可。

### 首次启动健康检查

第一次启动时，工具会自动检查所有配置是否正确，你会看到类似这样的报告：

| 检查项 | 状态 | 说明 |
|--------|------|------|
| Python 版本 | 通过 | 版本符合要求 |
| 浏览器引擎 | 通过 | Playwright 已安装 |
| 数据库 | 通过 | 可以正常读写 |
| 闲鱼 Cookie | 通过/警告 | Cookie 是否有效 |
| AI 服务 | 通过/跳过 | API 密钥是否配置 |

如果某项显示"失败"或"警告"，按照提示信息修复即可。

## 7. 使用教程：仪表盘

仪表盘是你打开工具后看到的第一个页面，它展示了你店铺的整体运营状况。

### 页面上的数字含义

| 数字卡片 | 含义 |
|---------|------|
| 在售商品 | 你目前在闲鱼上架的商品总数 |
| 总浏览量 | 所有商品被别人看过的总次数 |
| 总想要数 | 有多少人点了"我想要" |
| 总营收 | 你已经卖出去赚到的总金额 |

### 快捷操作

在仪表盘上你可以直接：
- 点击 **"一键擦亮"** - 自动帮你擦亮所有商品
- 点击 **"生成日报"** - 查看今天的运营数据汇总

## 8. 使用教程：发布商品

### 发布一个商品

1. 在左侧菜单栏点击 **"商品发布"**
2. 选择 **"单个发布"** 模式
3. 填写商品信息：

| 填写项 | 必填？ | 说明 | 举例 |
|--------|--------|------|------|
| 商品名称 | 是 | 你要卖的东西叫什么 | iPhone 15 Pro |
| 商品分类 | 是 | 属于哪个类别 | 数码手机 |
| 售价 | 是 | 你想卖多少钱 | 5999 |
| 原价 | 否 | 当初买的时候多少钱 | 8999 |
| 成色 | 是 | 新旧程度 | 95新 |
| 出售原因 | 否 | 为什么要卖 | 换新手机了 |
| 商品特性 | 否 | 有什么亮点 | 256GB、国行、配件齐全 |

4. 上传商品图片（最多 9 张）
   - 点击上传区域，选择电脑上的图片
   - 支持 JPG、PNG 格式
   - 建议每张图片不超过 5MB
5. 如果你配置了 AI 密钥，可以勾选：
   - **"AI 生成标题"** - 系统会根据你填写的信息自动写一个吸引人的标题
   - **"AI 生成描述"** - 系统会自动写一段商品描述
6. 确认所有信息无误后，点击 **"立即发布"**
7. 等待几秒钟，系统会显示发布结果

### 批量发布多个商品

如果你有很多商品要发布：

1. 选择 **"批量发布"** 模式
2. 下载商品信息模板文件
3. 用 Excel 或 WPS 打开模板，按照格式填写每个商品的信息
4. 保存文件后，上传到工具中
5. 设置发布间隔（建议 5-10 秒，太快容易被闲鱼限制）
6. 点击 **"开始批量发布"**

## 9. 使用教程：店铺运营（擦亮/调价/上下架）

点击左侧菜单栏的 **"运营管理"** 进入。

### 什么是"擦亮"？

在闲鱼上，"擦亮"商品可以让商品重新出现在搜索结果的前面。
就像把你的商品从货架后面拿到最显眼的位置。
每个商品每天可以免费擦亮一次。

### 批量擦亮

1. 设置要擦亮的商品数量（比如 50 个）
2. 设置操作间隔（建议 3-6 秒，太快会被闲鱼察觉）
3. 点击 **"开始批量擦亮"**
4. 工具会自动一个一个帮你擦亮
5. 完成后会显示结果：成功多少个、失败多少个

### 调整价格

**单个调价：**
1. 选择要调整价格的商品
2. 输入新价格
3. 点击 **"更新价格"**

**批量调价：**
1. 下载价格调整模板
2. 在模板中填写商品和新价格
3. 上传后执行

### 下架和上架

- **下架商品**：选中商品，点击"下架"，商品将从闲鱼上消失
- **重新上架**：选中已下架的商品，点击"重新上架"，商品会重新出现

## 10. 使用教程：账号管理

如果你有多个闲鱼账号，可以在这里统一管理。

点击左侧菜单栏的 **"账号管理"** 进入。

### 添加新账号

1. 点击 **"添加账号"**
2. 填写：
   - 账号名称：给这个账号起个名字，方便你区分（比如"主账号"、"副账号"）
   - 优先级：填数字 1、2、3...，数字越小越优先使用
   - Cookie：这个账号的 Cookie（获取方法见 [第二步](#4-第二步获取闲鱼登录凭证cookie)）
3. 点击 **"添加"** 保存

### 管理已有账号

对于每个账号，你可以：
- **启用/禁用**：暂时不用某个账号时可以禁用
- **设为当前**：指定工具当前使用哪个账号操作
- **刷新 Cookie**：Cookie 过期后，粘贴新的 Cookie
- **查看健康度**：检查账号运行是否正常

> 工具会自动加密存储你的 Cookie，不用担心泄露。

## 11. 使用教程：数据分析与报表

点击左侧菜单栏的 **"数据分析"** 进入。

### 查看报表

1. 选择报表类型：
   - **日报**：今天的数据
   - **周报**：最近 7 天的数据
   - **月报**：最近 30 天的数据
2. 点击 **"生成报表"**
3. 报表会显示：
   - 新发布了多少商品
   - 被浏览了多少次
   - 有多少人想要
   - 成交了多少笔
   - 收入多少

### 查看趋势

1. 选择你关心的指标（浏览量、想要数、成交量、收入）
2. 选择时间范围（7 天 / 30 天 / 90 天）
3. 系统会画出一条趋势曲线，让你直观看到数据变化

### 导出数据

如果你需要把数据保存到电脑上（比如交给老板看）：
1. 选择导出类型和格式（CSV 或 JSON）
2. 点击 **"导出数据"**
3. 文件会保存到 `data` 文件夹中

## 12. 使用教程：定时任务

你可以设置工具自动在每天的固定时间帮你做事情。

在 **"账号管理"** 页面中，切换到 **"定时任务"** 标签。

### 创建定时任务

1. 点击 **"创建任务"**
2. 填写：
   - 任务名称：比如"每天早上9点擦亮"
   - 任务类型：
     - 自动擦亮 - 帮你自动擦亮商品
     - 数据采集 - 自动记录你的运营数据
     - 健康检查 - 自动检查系统是否正常
   - 执行时间：选择什么时候执行
3. 点击 **"保存"**

### 常用的执行时间设置

| 你想要 | 怎么设置 |
|--------|---------|
| 每天早上 9 点 | `0 9 * * *` |
| 每天早上 9 点和下午 3 点 | `0 9,15 * * *` |
| 每 4 小时一次 | `0 */4 * * *` |
| 每周一早上 9 点 | `0 9 * * 1` |

> 上面那串看起来像密码的文字叫"Cron 表达式"，你只需要复制对应的那串到执行时间里就行。

## 13. 日常维护

### 每天要做的事

其实什么都不用做。如果你设置了定时任务，工具会自动运行。
你只需要偶尔打开网页看看仪表盘，了解运营情况。

### 每周建议做的事

1. **检查 Cookie 是否过期**
   - 打开"账号管理"页面
   - 如果看到"Cookie 已过期"的提示，需要重新获取并更新
   
2. **查看周报**
   - 打开"数据分析"，生成周报，了解本周的运营情况

### Cookie 过期了怎么办？

Cookie 过期是最常见的问题。过期后工具无法操作闲鱼。

**解决方法：**
1. 重新打开浏览器，登录闲鱼网页版
2. 按照 [第二步](#4-第二步获取闲鱼登录凭证cookie) 重新获取 Cookie
3. 在"账号管理"页面，找到对应账号，点击 **"刷新 Cookie"**
4. 粘贴新的 Cookie 并保存

### 如何关闭工具

**Windows：** 关闭那个黑色的命令窗口即可

**macOS：** 在终端中按 `Ctrl + C`

**Docker：** 执行 `docker compose down`

### 如何重新启动

关闭后，按照 [第四步](#6-第四步启动工具) 重新启动即可。你之前的所有数据和配置都会保留。

## 14. 遇到问题怎么办？

### 问题 1：安装脚本运行失败

**可能的原因：** Python 或 Node.js 没有正确安装。

**解决方法：**
1. 重新按照 [3.1](#31-安装-python工具运行需要它) 和 [3.2](#32-安装-nodejs网页界面需要它) 安装
2. 安装 Python 时一定要勾选 **"Add Python to PATH"**
3. 安装完成后，重启电脑，再运行安装脚本

### 问题 2：打开网页后显示空白或报错

**解决方法：**
1. 等待 10 秒钟再刷新页面（工具可能还在启动中）
2. 检查启动窗口中是否有红色的错误信息
3. 关闭工具后重新启动

### 问题 3：发布商品失败

**可能的原因和解决方法：**

| 可能原因 | 解决方法 |
|---------|---------|
| Cookie 过期 | 重新获取 Cookie（见 [日常维护](#cookie-过期了怎么办)） |
| 商品信息不完整 | 检查是否填写了所有必填项 |
| 图片格式不对 | 使用 JPG 或 PNG 格式的图片 |
| 图片太大 | 每张图片建议不超过 5MB |
| 闲鱼临时限制 | 等待几分钟后重试 |

### 问题 4：AI 生成标题/描述失败

**解决方法：**
1. 检查 `.env` 文件中的 API 密钥是否填写正确
2. 确认你的 AI 账户有余额
3. 检查电脑能否正常上网
4. 实在不行的话，关闭 AI 生成功能，手动填写标题和描述

### 问题 5：批量操作时卡住不动

**解决方法：**
1. 耐心等待，批量操作需要时间（每个商品间隔 3-10 秒）
2. 如果等了很久仍然没反应，刷新网页重试
3. 操作间隔不要设置太短（至少 3 秒以上）

### 问题 6：如何查看错误日志？

如果你需要请别人帮忙排查问题，可以把日志文件发给他：

1. 打开 `xianyu-openclaw-main` 文件夹
2. 进入 `logs` 文件夹
3. 找到日期最新的文件，这就是最新的日志

## 15. 名词解释

| 名词 | 通俗解释 |
|------|---------|
| Cookie | 你登录闲鱼后浏览器保存的"通行证"，工具用它来代替你操作 |
| API Key（API 密钥） | AI 服务的"钥匙"，有了它才能让 AI 帮你写文案 |
| 擦亮 | 闲鱼的一个功能，刷新商品让它排名更靠前，每天免费一次 |
| 仪表盘 | 一个展示所有关键数据的汇总页面 |
| 日报/周报/月报 | 按天/周/月汇总的运营数据报告 |
| 定时任务 | 设定好时间后，工具在那个时间自动执行指定操作 |
| 下架 | 让商品从闲鱼上消失，别人搜不到也看不到 |
| 上架 | 把商品放到闲鱼上，别人可以看到和购买 |
| Docker | 一种把软件打包好的技术，方便在服务器上运行（普通用户不需要了解） |
| 终端 | macOS 上的一个程序，用来输入命令让电脑执行任务（类似 Windows 的命令提示符） |
| Cron 表达式 | 一种表示时间周期的格式，用来告诉工具"每隔多久执行一次" |

## 安全提示

1. **不要把你的 Cookie 和 API 密钥告诉任何人** — 它们相当于你的密码
2. **定期更新 Cookie** — 过期的 Cookie 无法使用
3. **不要在公共电脑上使用本工具** — 你的账号信息可能被别人看到
4. **操作间隔不要太短** — 太频繁的自动操作可能导致闲鱼限制你的账号
5. **遵守闲鱼平台规则** — 本工具仅用于提升运营效率，请合理使用
